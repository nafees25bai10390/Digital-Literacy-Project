Project Overview
This Digital Literacy Project was completed as part of the CSE0001 course. The objective of this project is to develop awareness about digital tools, online safety, professional communication, and cybercrime prevention.

The project consists of five tasks that focus on different aspects of digital literacy. All tasks are organized in this repository along with the project report and supporting materials.
